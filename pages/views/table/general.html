主要用于展示大量结构化数据。

  • 当有大量结构化的数据需要展现时；
  • 当需要对数据进行排序、搜索、分页、自定义操作等复杂行为时。

代码示例

基础用法

表格的最简单用法。默认自带斑马纹，可通过表格参数striped控制。默认自带分页，可通过表格参数pagination控制。


<table id="table1"></table>
                
$('#table1').table({
    pagination: false,//取消分页
    columns: [{
        field: 'id',
        title: 'Item ID'
    }, {
        field: 'name',
        title: 'Item Name'
    }, {
        field: 'price',
        title: 'Item Price'
    }],
    data: [{
        id: 1,
        name: 'Item 1',
        price: '¥ 1'
    }, {
        id: 2,
        name: 'Item 2',
        price: '¥ 2'
    }, {
        id: 3,
        name: 'Item 3',
        price: '¥ 3'
    }]
});

表格formatter及事件绑定

columns参数formatter配合events使用实现单个或多个事件绑定。 


<table id="table6"></table>
                  
$('#table6').table({
    pagination: false,
    columns: [{
        checkbox: true
    }, {
        field: 'id',
        title: 'Item ID'
    }, {
        field: 'name',
        title: 'Item Name'
    }, {
        field: 'price',
        title: 'Item Price',
        events: {
            'click .myclass': function(e, value, row, index){
                e.stopPropagation();//防勾选
                $.alert(value);
            }
        },
        formatter: function(value, row, index, field){
            var color;
            switch(value){
                case '2': color = 'green';
                break;
                case '0': color = 'red';
                break;
                default: color = 'blue';
                break;
            }
            return '<span class="myclass" style="color:'+color+'">¥ '+value+'</span>';
        }
    }],
    data:[
        {
            "id": 0,
            "name": "Item 0",
            "price": '0'
        },
        {
            "id": 1,
            "name": "Item 2",
            "price": '1'
        },
        {
            "id": 2,
            "name": "Item 1",
            "price": '2'
        }
    ]
});
列定宽表格（非响应式表格）

可通过添加参数classes: 'si-table-fixed'或者直接给table添加si-table-fixed的class。

定宽表格超出宽度将会被省略。


<table id="table7"></table>
                  
$('#table7').table({
    classes: 'si-table-fixed',
    pagination: false,
    columns: [{...}],
    data: [{...}]
});

//或者
<table class="si-table-fixed" id="table6"></table>
                  
$('#table6').table({
    pagination: false,
    columns: [{...}],
    data: [{...}]
});
