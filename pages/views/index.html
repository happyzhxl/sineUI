SineUI，是基于jQuery的开源 UI 组件库，引入bootstrap3的响应式设计，既面向PC端后台系统，也可完成移动端的屏幕适配。

支持环境
现代浏览器和 IE9 及以上
浏览器引入 

<!-- 最新版本的 SineUI CSS 文件 -->
<link rel="stylesheet" type="text/css" href="https://www.sineui.info/dist/css/sine-1.2.6.min.css">

<!-- 可选的 SineUI 主题文件（与基础样式有且只有一个） -->
<link rel="stylesheet" type="text/css" href="https://www.sineui.info/dist/css/sine-info-1.2.6.min.css">

<!-- 最新版本的 SineUI JS 文件 -->
<script type="text/javascript" src="https://www.sineui.info/dist/js/sine-1.2.6.min.js"></script>
                    
//初始化Sine，可配置全局路由、主题等
new Sine();
